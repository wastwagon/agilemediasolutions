HOW TO VIEW THE SITE (so Careers and other pages work)
=====================================================

If you see "127.0.0.1 refused to connect" when clicking Careers, no local server is running. Use one of these:

OPTION 1 - Easiest (run the server)
-----------------------------------
1. Double-click: Start-Site.bat
2. Your browser will open to the home page.
3. Click "Careers" in the menu or "Explore Careers" on the page.
   The Careers page will load correctly.

OPTION 2 - Open files directly (no server)
------------------------------------------
1. In File Explorer go to this folder: agile-media-website
2. Double-click index.html (opens in your browser).
3. Click "Careers" or "Explore Careers".
   careers.html will open in the same folder.

OPTION 3 - PowerShell server
----------------------------
1. Right-click start-server.ps1 → Run with PowerShell.
2. In browser open: http://localhost:8080/index.html
3. Then click Careers.

Note: Always open the site from inside the agile-media-website folder so that links like careers.html, services.html, etc. work.
